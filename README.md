# VisionBot  

基于视觉语言模型（VLM）的智能电脑自动化操作工具。

## 项目说明

本项目是对 [thenextagent](https://github.com/qingningLime/thenextagent) 的重构版本，几乎将原有的代码完全重构，但模型定义层还是参考上面的项目，在原有功能基础上进行了大幅扩展和优化，支持更多应用场景和更智能的操作体验。
我们采用了pyautogui，这种操作方式支持所有系统，能力有限目前仅仅是适配了几款应用，欢迎提issue和star，对类似抖音和快手进行测试型应用级优化（也就是通过rag里写入特殊应用特殊方法）。可以一次执行多步操作，例如打开抖音给船长d的第一个视频点一个赞，会打开浏览器-->找到抖音网页版-->找到用户船长d，给他点赞

## 功能特点
- 通过VLM模型实时分析屏幕内容，自动执行鼠标点击、键盘输入等操作
- 注重隐私保护，需要用户操作的敏感操作，模型会自动暂停
- 支持多种应用场景，目前在prompt中支持了RAG里面有关于抖音和快手的操作，正在验证中
- 操作历史记录和智能坐标调整

## 使用方法
1. 配置 `model_config.json` 文件，填入你的API密钥，仓库里的已经修改过了，只是一个样例。
2. 运行 `python gui.py` 启动图形界面
3. 添加任务并开始执行
